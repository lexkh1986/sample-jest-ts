# Project Description

This project is a learning material for nodejs automation test. Designed by Lex.Khuat during his first time self-learning nodejs.

## Requirements
- NodeJS

## Installation
```bash or powershell
npm install -g yarn
```

## Usage

```Test run
yarn test
```
